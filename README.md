# GEO优化平台

基于 Vue 3 + Element Plus + Express + TypeScript 的完整前后端项目。

## 快速开始

### 前端
```bash
cd frontend
npm install
npm run dev
# 访问 http://localhost:5173
```

### 后端
```bash
cd backend
npm install
npm run dev
# 访问 http://localhost:3000
```

## 技术栈

- 前端: Vue 3 + Vite + Element Plus + TypeScript
- 后端: Express + TypeScript + Prisma + PostgreSQL

## 目录结构

```
geo-platform/
├── frontend/          # Vue 3 前端
├── backend/           # Express 后端
└── docker-compose.yml
```
